## Árbol del amor

> <b>Modificando la configuración de config.js, puedes crear tu propio Árbol del amor. Si te gusta, ¡no dudes en bifurcar o marcar el proyecto con una estrella!</b>

<img src="https://github.com/AJLoveChina/loveBalloon/blob/master/static/github-star.png" />

### Explicación de config.js
```texto
var config = {
    // El número de líneas a continuación no tiene por qué ser siete; puedes ajustarlo a cuatro, dos o menos.
    líneas: [
        «Para el amor de mi vida Yasmina »,
        «Si pudiera elegir un lugar »,
        «seguro, seria a tu lado»,
        «Cuanto mas tiempo estoy »,
        «contigo mas te amo»,
        «Mi querido amorcito, eres mi único amor »,
        «Te amaré para siempre...»,
    ],
    // Mi amor comenzo por ti hace....
    // El formato debe coincidir exactamente con el formato de tiempo que se indica a continuación.
    // Año-Mes-Día Hora:Minuto:Segundo
    fecha: «2017-09-07 00:00:0»,
    // Nombres de ambas partes
    nombres: [«Chucurrumi», «Mi tesorote hermsote»]
}

Consejo: Para cambiar la música de fondo, simplemente sustituye `index_files/music.mp3` por tu propia pista, pero no modifiques el nombre del archivo.

Si prefieres no tener música, simplemente elimina el archivo music.mp3.
